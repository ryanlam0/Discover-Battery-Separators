Current Li-ion battery separators are vulnerable to thermal and mechanical stress, leading to thermal runaway and fire.
new polymer separators with superior thermal/mechanical stability, but experimentally testing every polymer is expensive and slow. 
This project trained a KNN classifier on 39 known separator polymers to predict safety from 6 material properties (3 thermal, 3 mechanical).
The model then screened 101 untested polymers to identify 5 strong candidates. 
The model achieves cross-validated safe precision = 0.852.
